TsutaAI Desktop App

デスクトップアプリは、今日のタスク選択・作業ログ・カンバン更新・日報送信までをパソコン1台で完結させるための常駐アシスタントです。ここでは実際の画面に沿って、操作手順・便利な使い方・トラブル時のヒントを丁寧に解説します。

レイアウトとログイン

ログイン手順

起動するとメールアドレスとパスワードの入力画面が表示されます。認証に成功すると、直前のセッション情報を引き継いだ状態でダッシュボードが開きます。入力漏れや誤入力は赤字で即時フィードバックされるため安心です。

  • 自動サインイン: Windowsの資格情報マネージャーに暗号化して保存でき、次回起動時は数秒でログインが完了します。
  • 多要素認証: 必要な組織ではワンタイムコード入力にも対応。専用ダイアログが自動表示されるため手順で迷いません。
  • 接続エラー時: ネットワークが不安定でも入力値は保持され、［再試行］を押すだけでリカバリーできます。
ヒント: 端末を家族と共用する場合は、自動ログインをオフにして毎回サインインする運用が推奨です。

初回セットアップと画面構成

初回ログイン直後はセットアップウィザードが一度だけ表示され、作業フォルダ・API URL・監視モードなどを案内します。設定完了後の画面構成は「サイドバー」「上部ヘッダー」「メインコンテンツ」の3区域で統一されており、どの画面でも迷わず遷移できます。

  • 作業フォルダ: 日報控え・ログ・一時ファイルが保存される場所です。OneDriveや社内NASを指定するとバックアップになります。
  • API接続テスト: ［接続確認］ボタンで /health エンドポイントを呼び出し、成功可否をモーダルで表示します。
  • 監視モード選択: 勤務時間のみ操作ログを取得するスケジュール設定が可能で、プライベート時間の記録を避けられます。

ダッシュボード

今日のタスクとクイック操作

本日割り当てられたタスクがカード形式で表示されます。チェックを付けて［作業開始］を押すと常駐ウィジェットが起動し、タイムトラッキングがスタートします。AIが締切や依存変更を検知すると、カード上にバッジとメッセージを表示して優先順位を教えてくれます。

  • 複数選択: 複数タスクをまとめて選択すると、ウィジェット上で完了した順に次タスクへ自動で切り替わります。
  • メニューバー: 右上のメニューから設定・カンバン・日報・ログアウトへワンクリックで移動できます。
  • ショートカット: 各カードの「詳細」リンクでタスク詳細ダイアログを開き、担当者変更やコメント追記が可能です。

おすすめルーティン

朝イチでダッシュボードを確認し、AIが「優先」と判定したカードから着手します。昼前にいったんウィジェットを一時停止し、午後の予定を再確認。終業前は「未完了」フィルタで残タスクを洗い出し、ゼロになったら日報画面へ進む――このサイクルで進捗漏れを防げます。

  • 昼休み前にウィジェットを停止すると、午前・午後のログがきれいに分割され、日報の見やすさが向上します。
  • 「日報」ボタンは途中下書きを残したい時にも利用でき、作業の区切りごとに感想をメモしておくとAIの精度が高まります。

常駐ウィジェット

リアルタイム記録と操作

画面右下に常駐し、現在のタスク名・予定時間・経過時間・進捗バー・操作回数をリアルタイム表示します。完了ボタンを押した瞬間に作業ログがバックエンドへ送信され、次のタスクに自動的に切り替わります。

  • 一時停止／再開: 予定外の会議や休憩ではワンクリックでログを停止。再開すると同じタスクで記録が継続します。
  • 並べ替え: 複数タスクを選択している場合、ウィジェット内でドラッグして優先順位を変更できます。
  • 自動保存: ネットワーク切断時はローカルに退避し、復旧後に自動送信します。
  • 操作量の可視化: キーボード／マウス入力の累計が常に表示され、集中度の目安になります。

よくある質問と対処

  • ウィジェットを誤って閉じても、ダッシュボードから［ウィジェットを開く］を押せば状態を保ったまま再表示できます。
  • 長時間アイドル状態になると通知が出て、ログの取りこぼしを防ぎます。通知をクリックするとそのまま再開可能です。
  • ノートPCを移動させる際は一時停止してからスリープに入れると、ログが午前／午後で分かれてレポートが読みやすくなります。

カンバンボード

リアルタイムでの状況共有

デスクトップ版カンバンはWeb版と同じデータを共有し、列の追加・カード移動・担当者変更が即時にチームへ反映されます。開発メンバーがWebブラウザを開かなくても、日中の軽微な調整をアプリだけで完結できます。

  • ドラッグ操作: カードを列間で移動すると、WBSの状態やダッシュボードの並びも自動更新されます。
  • カード詳細: ダブルクリックでコメント、添付ファイル、Git差分を確認でき、デスクトップでそのまま修正作業に入れます。
  • スイムレーン切替: 優先度・担当者・リリース単位などの横方向ラベルを自由に追加できます。

日報作成

AI下書きの確認

ウィジェットで収集した作業ログやGit差分から、AIが「要約／達成事項／課題／次のアクション」を自動生成します。文章を修正すると即座に「編集済み」タグが付き、加筆箇所が分かります。

  • 再生成ボタン: ログを追加した後や書き直したい場合に押すと、最新情報で草案を再作成します（「再生成」というボタン名が画面上にも表示されています）。
  • 自己評価スライダー: 気分・満足度を入力すると、Web管理画面のメンタルダッシュボードにも反映されます。
  • AIフィードバック: 送信後には励ましや改善案が別枠に表示され、翌日の行動に役立ちます。

送信と控えの管理

［日報を送信］を押すとバックエンドに保存され、リーダーへ通知されます。ローカルの作業フォルダにも控えが自動保存され、オフラインでも参照可能です。

  • 未送信のままアプリを閉じようとすると確認ダイアログが表示され、書き忘れを防ぎます。
  • 送信後に修正したい場合は、Web管理画面で対象レポートを開き直して編集できます。

設定

接続とフォルダ

APIエンドポイントや作業フォルダなど日常的に変更する可能性がある項目はこの画面にまとまっています。項目ごとの説明とおすすめ設定値をチェックしながら調整できます。

  • API URL: 会社環境（例：本番／検証）ごとに切り替え可能。［接続テスト］ボタンで疎通を確認できます。
  • 作業フォルダ選択: ボタン一つでエクスプローラーが開き、保存先を直感的に変更できます。変更後はアプリ再起動なしで反映されます。
  • プロキシ設定: 社内ネットワークを経由する場合でも、ユーザー名とパスワードを登録しておけば自動接続されます。

監視・通知・集中モード

勤務形態に合わせてログ取得の範囲や通知方法をきめ細かく設定できます。集中したい時間帯は通知を減らし、チームにフォローしてほしいときはアラートを増やすなど、運用に合わせた調整が可能です。

  • 活動ログON/OFF: キーボード／マウスの記録を停止したい場合はチェックを外すだけ。プライバシーが気になるときに便利です。
  • 通知チャンネル: デスクトップ通知・メール通知・アプリ内通知のオン／オフを個別に切り替えられます。
  • 集中モード: 一時的に新規タスク割り当てを控えてほしい週は集中モードをオンにし、理由や期間を記入するとAIが他メンバーの稼働を優先するよう調整します。

休暇・稼働調整

休暇や在宅勤務の予定をアプリから登録すると、Web管理画面やWBSにも自動反映されます。AIがリスケ案を作成し、関係者へ通知してくれるため、手作業での連絡が不要です。

  • 休暇登録: 日付とメモを入力するだけで、当日のタスクはダッシュボードから除外されます。
  • 代替日入力: 作業の持ち越し先を指定すると、AIが新しいスケジュール案を生成し、承認フローまでサポートします。
  • 稼働率スライダー: 数週間単位で稼働率を上げ下げすると、AIが他メンバーへのタスク配分を調整します。

アップデートとサポート

アプリのバージョン確認・自動更新・ログのエクスポートなど、運用担当者向けのメニューもこの画面に含まれています。トラブル時はサポート情報をまとめて取得できます。

  • バージョン確認: 現在のバージョンと最新バージョンを表示し、ワンクリックで更新できます。
  • ログエクスポート: サポート依頼時に必要なアプリログをZIPでまとめてダウンロードできます。
  • 問い合わせリンク: ヘルプリクエスト画面へ遷移し、開発チームとチャットを開始できます。

AIサポート機能

自動日報ジェネレーター

作業ログとGit差分をAIが整理し、職種別テンプレートに沿って日報下書きを提示します。テクニカル職ならコード差分やビルド結果、PMならリスク項目が強調されます。

アクティビティ解析

ActivityMonitorServiceとSystemPerformanceMonitorがCPU・メモリ負荷、入力頻度、アプリ使用状況を継続的に収集し、集中度レポートとして表示します。長時間アイドル状態が続く場合はウィジェット上でリマインドされます。

メンタルケア

日報の自己評価やチャット内容からストレスサインを検出し、色分けされたスコアと推奨アクション（例：短い休憩、一人作業の増加）が提示されます。低スコアが続くとマネージャーにも通知され、早期フォローにつながります。

ヘルプリクエスト連携

画面右上のベルアイコンからヘルプリクエスト一覧を開くと、サポート待ちのタスクが把握できます。［対応する］を押すだけで自分が担当として登録され、進捗はAIが追跡します。
